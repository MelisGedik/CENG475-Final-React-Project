# Project-12: Movie Recommendation Platform (CENG475)

A full-stack movie platform that provides **personalized movie recommendations** based on user preferences and watch history. Users can browse a movie catalog, search & filter movies, view details, and (optionally) rate/review watched movies. Admin users can manage the movie database and user roles.

## Objective
To provide personalized movie recommendations based on user preferences.

## Features
- **Movie Database (Catalog):** Movies with genre tags, descriptions, ratings, and details
- **Recommendations:** Suggestions based on user watch history and preferences
- **Rating & Reviews:** Users can rate/review movies they watched (if enabled in the project)
- **Search & Filter:** Search by title and filter by genre, rating, release year
- **Auth (Sign up / Sign in):** JWT-based authentication
- **User Roles:** At least **2 roles** (e.g., `user` and `admin`)
- **Admin Panel:** Admins can manage movies, users, and recommendation-related settings

## Tech Stack
- **Frontend:** React + Vite, HTML, CSS, JavaScript
- **Backend:** Node.js + Express
- **Database:** MySQL (tables + CRUD)
- **Auth:** JWT (token-based), password hashing
